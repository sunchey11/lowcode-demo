## 使用

### 注册插件
```jsx
import { plugins } from '@alilc/lowcode-engine';
import LowcodePluginAndersPlugin from 'lowcode-plugin-anders_plugin';

// 注册到引擎
plugins.register(LowcodePluginAndersPlugin);
```
